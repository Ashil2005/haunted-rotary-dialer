# Audio Assets

Place your audio files here:

- `click.wav` - Short click sound for each notch (50-100ms)
- `release.wav` - Spring release sound (300-500ms)

## Recommended Sources

1. **Record from real rotary phone** (most authentic)
2. **Free sound libraries:**
   - Freesound.org
   - Zapsplat.com
   - BBC Sound Effects

## Audio Specifications

- Format: WAV or MP3
- Sample Rate: 44.1kHz
- Bit Depth: 16-bit
- Channels: Mono
- Click duration: 50-100ms
- Release duration: 300-500ms

## Temporary Solution

The app will work without audio files (errors are caught gracefully).
Add real audio files for the full experience.
